This project analyzes smoking patterns in UK culture through practical data visualization exercises.

## Dataset
- **Source**: Smoking dataset
- **Focus**: UK cultural smoking trends

## Analysis
The analysis pivots the smoke column and examines relationships with other dataset variables.
